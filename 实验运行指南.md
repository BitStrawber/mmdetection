## 二、数据准备

### 2.1 COCO数据集

```bash
# 下载COCO 2017训练集（约18万张）
# https://cocodataset.org/#download

# 目录结构:
# /path/to/coco/
# ├── annotations/
# │   └── instances_train2017.json
# └── train2017/
#     ├── 000000.jpg
#     ├── 000001.jpg
#     └── ...
```

### 2.2 抽样50k张

```bash
cd UWNR
python sample_coco.py \
    --ann /path/to/coco/annotations/instances_train2017.json \
    --img-dir /path/to/coco/train2017 \
    --output-dir /path/to/coco_uwnr \
    --num 50000
```

### 2.3 UWNR转换水下图像（双GPU并行）

```bash
# 终端1: GPU6, 处理前25000张
python convert_coco_uwnr.py \
    --ann /path/to/coco_uwnr/annotations/instances_train50000.json \
    --img-dir /path/to/coco_uwnr/images \
    --output-dir /path/to/coco_uwnr \
    --uwnr-dir ./ \
    --uwnr-model ./UWNR.pk \
    --gpu 6 \
    --start 0 --end 25000 &

# 终端2: GPU7, 处理后25000张
python convert_coco_uwnr.py \
    --ann /path/to/coco_uwnr/annotations/instances_train50000.json \
    --img-dir /path/to/coco_uwnr/images \
    --output-dir /path/to/coco_uwnr \
    --uwnr-dir ./ \
    --uwnr-model ./UWNR.pk \
    --gpu 7 \
    --start 25000 --end 50000 &
```

转换完成后：
```
/path/to/coco_uwnr/
├── annotations/
│   └── instances_train50000.json
└── images/   # 50k张水下图像
```

## 三、实验配置

### 3.1 修改数据路径

```bash
cd mmdetection

# 替换路径占位符
sed -i 's|/path/to/data/|/你的实际路径/|g' \
    configs/cascade_rcnn/cascade-rcnn_r50_fpn_2x_ruod.py \
    configs/cascade_rcnn/cascade-rcnn_r50_fpn_2x_coco_uwnr.py \
    configs/cascade_rcnn/cascade-rcnn_r50_fpn_2x_ruod_uwnr_pretrain.py
```

需要配置的路径：
- `COCO_UWNR_DIR`: `/path/to/coco_uwnr/` （UWNR转换后的数据集）
- `RUOD_PIC_DIR`: `/path/to/RUOD_pic/` （RUOD图片）
- `RUOD_ANN_DIR`: `/path/to/RUOD_ANN/` （RUOD标注）

## 四、训练与测试

### 实验A：ImageNet预训练 + RUOD（对照组）

```bash
cd mmdetection

# 训练
python -m torch.distributed.launch --nproc_per_node=8 \
    tools/train.py \
    configs/cascade_rcnn/cascade-rcnn_r50_fpn_2x_ruod.py

# 测试
python tools/test.py \
    configs/cascade_rcnn/cascade-rcnn_r50_fpn_2x_ruod.py \
    work_dirs/cascade-rcnn_r50_fpn_2x_ruod/latest.pth \
    --eval bbox \
    --cfg-options model.init_cfg=None
```

### 实验B：COCO-UWNR预训练 + RUOD

```bash
cd mmdetection

# B-1: 在UWNR增强的COCO上训练
python -m torch.distributed.launch --nproc_per_node=8 \
    tools/train.py \
    configs/cascade_rcnn/cascade-rcnn_r50_fpn_2x_coco_uwnr.py

# B-2: 提取backbone权重（处理80类→10类不兼容）
python tools/extract_backbone.py \
    --checkpoint work_dirs/cascade-rcnn_r50_fpn_2x_coco_uwnr/latest.pth \
    --output work_dirs/cascade-rcnn_r50_fpn_2x_coco_uwnr/backbone_only.pth

# B-3: 微调RUOD
python -m torch.distributed.launch --nproc_per_node=8 \
    tools/train.py \
    configs/cascade_rcnn/cascade-rcnn_r50_fpn_2x_ruod_uwnr_pretrain.py

# 测试
python tools/test.py \
    configs/cascade_rcnn/cascade-rcnn_r50_fpn_2x_ruod_uwnr_pretrain.py \
    work_dirs/cascade-rcnn_r50_fpn_2x_ruod_uwnr_pretrain/latest.pth \
    --eval bbox \
    --cfg-options model.init_cfg=None
```

## 六、常见问题

### Q: 转换图片很慢怎么办？
A: 用双GPU并行（如上所示），或者先转换少量图片测试流程。

### Q: 训练中断如何继续？
A: 使用 `--resume-from work_dirs/.../latest.pth` 恢复训练。

### Q: GPU显存不够怎么办？
A: 减小 `samples_per_gpu`，比如从2改为1，总BS从16变为8。

### Q: 如何查看训练日志？
A: `tail -f work_dirs/<config_name>/vis_data/scalars.json` 或 tensorboard。
